JavaScript Array Methods

Array.map();

The array.map() method is used to change the elements in an array in a way specified by the user, and returns a new array with the updated elements.

Map takes 2 arguments, only the first is required, it can be an expression or a function that returns a value, which is added to the new array.

Don't use map if:

  • Your callback function doesn't return a value.
  • You don't need to return an array.
  • Use forEach() instead.

Syntax:

          arrayObject.map(callback[,contextObject]);

Callback Syntax:

  • currentElement: element that is passed in from the array
  • index: the current index of the currentElement
  • array: the original array that is passed through map
          function callback(currentElement, index, array) {...}

Examples:

        let numArray = [];
        

        const getSqrt = numArray.map((num) => Math.sqrt(num));
        
          let tempsArray = [];
          

          function convertToC(fahr) {
            let cel = (fahr - 32) / 1.8;

            return cel.toFixed(2);
          }

          let celsius = tempsArray.map(convertToC);
        
          let skipArray = [];

          const skipNum = skipArray.map((num) => {
            let temp = (num * 1.8 + 32).toFixed(2);

            if (temp <= 85 && temp >= 65) {
              return temp;
            }

            return "N/A";
          });

Array.filter();

Array.filter() takes an array and returns a filtered array based on a condition provided by the user.

Almost identical to map, filter takes 2 arguments, only the first, which is a callback function, is required.

If you want to filter the array and make changes to it at the same time, map() may be a better option.

Syntax:

          arrayObject.filter(callback,contextObject);

Callback Syntax:

  • currentElement: element that is passed in from the array
  • index: the current index of the currentElement
  • array: the original array that is passed through filter
          function callback(currentElement, index, array) {...}

Examples:

        let unfilteredArray = [];
        

        let filterArray = unfilteredArray.filter((num) => num > 75);
        
        let employees = [
          { name: "Bob", sold: 306 },
          { name: "Sally", sold: 472 },
          { name: "Joey", sold: 193 },
          { name: "Rupert", sold: 388 },
          {}
        ];
        

        // Finding top sellers with for loop
        let topSellers = [];

        for (let i = 0; i < employees.length; i++) {
          if (employees[i].sold > 300) {
            topSellers.push(employees[i]);
          }
        }
 
        // Finding top sellers using filter
        let filterTopSellers = employees.filter(function (merch) {
          return merch.sold > 300;
        });

Array.reduce();

Array.reduce() takes an array and condenses it to a single value after performing a requested operation.

Reduce() has 2 arguments, the callback function and an optional initialValue.

Any updates made to the array after iteration begins will not be counted by reduce (i.e.: append).

Items that are deleted from the array before their turn in the sequence will not be counted by reduce().

Syntax:

          arrayObject.reduce(callback[,initialValue]);

Callback Syntax:

  • initialValue: if supplied, this is the value that is iterated over by reduce()
  • previousValue (prev): value at array[0] by default, but can be supplied by user
  • currentValue (current): value at array[0] if user supplies initialValue, otherwise value at array[1]
  • index: the current index of the currentElement
  • array: the original array that will be reduced
          function callback(prev, current, index, array) {...}

Examples:

          let smallestNum = (a, b) => Math.min(a, b);

          console.log(nums.reduce(smallestNum));
        
          let purchasedItems = [
            {
              name: "apple",

              qty: 5,
            },
            {
              name: "orange",
              price: 0.99,
              qty: 4,
            },
            {
              name: "banana",
              price: 1.49,
              qty: 2,
            },
          ];

Array.every();

The every() method is used to check to see if all elements in an array meet a certain condition. If they do, it returns true, if not it returns false.

Every() does not change the original array and it does not return an updated array.

Like other Array methods every() takes a mandatory callback function and the optional thisArg, which allows thisArg to reference itself from inside the callback function with the "this" value.

The callback syntax is similar to map() and filter(), accepting three arguments, the current element, the index, and the array itself.

When called, every() will run until it returns a false (falsy) value or reaches the end of the array.

Syntax:

          arrayObject.every(callback[,thisArg]);

Callback Syntax:

          function callback(currentElement, index, array) {...}

Array.some();

Array.some() is similar to every(); however, it checks to see if at least one item in the array meets the condition, if so, it returns true.

The some() method determines the range of elements to be checked before running, so it will not check any elements that are added after it is invoked.

When called, some() will run until it returns a true (truthy) value, or reaches the end of the array.

If some() is called on an empty array it will always return false.

Syntax:

          arrayObject.some(callback[,thisArg]);

Callback Syntax:

          function callback(currentElement[[, index], array]) {...}

Array.every(); and Array.some(); continued...

Examples:

          let allPos = [1, 13, 4, 75, 8, 629, 2];
          let oneLess = [];

          // every
          const everyA = allPos.every((number) => number > 0); 
          // returns true
          const everyB = oneLess.every((number) => number > 0); 
          // returns false

          // some
          const someA = allPos.some((number) => number > 0); 
          // returns true
          const someB = oneLess.some((number) => number > 0); 
          // returns true
        
          const isIncluded = (set1, set2) 
            => set2.every((item) 
            => set1.includes(item));

          // Works on all types of data
          console.log(
            isIncluded(
              [6, 20, 5, 77, 22, 3, 12, 9], 
              [6, 20, 22]
            )
          ); // true

          console.log("Every results: ");
          console.log(
            isIncluded(
              ["a", "b", "c", "d", "e", "f"],
              ["e", "a", "d"]
            )
          ); // true
          console.log(
            isIncluded(
              ["a", "b", "c", "d", "e", "f"], 
              ["f", "h", "g"]
            )
          ); // false

          const includedOnce = (set1, set2) 
            => set2.some((item) 
            => set1.includes(item));

          console.log("Some results: ");
          console.log(
            includedOnce(
              ["a", "b", "c", "d", "e", "f"], 
              ["e", "a", "d"]
            )
          ); // true
          console.log(
            includedOnce(
              ["a", "b", "c", "d", "e", "f"],

            )
          ); // true

          const rangeA = {
            min: 94,
            max: 100,
          };

Array.sort();

The sort() method takes the elements of an array and sorts them in ascending order based on UTF-16 code values.

Unlike the other methods we've covered sort() does not make a new sorted array, instead it sorts and updates the current array.

Sort() takes an optional compare function.

Syntax:

          arrayObject.sort(compareFunction);

The compare() function takes two arguments, sort order is determined by what compare returns:

  • If compare(a, b) < 0; a is listed before b
  • If compare(a, b) > 0; b is listed before a
  • If compare(a, b) === 0; the order remains unchanged

Essentially, if the compare function is not provided, all values are converted to strings and sorted by string comparison. So, ['b', 'e', 'd', 'c', 'a'] becomes ['a', 'b', 'c', 'd', 'e'] and [3, 5, 1, 30, 11] becomes ["1", "11", "3", "30", "5"].

If you have to sort words with non-ASCII special characters like é, è, etc., you can use the localeCompare() function.

Compare() Syntax:

          function compare(a, b) {...}

Examples:

          let letters = ['b', 'e', 'd', 'c', 'a'];
          let numbers = [3, 5, 1, 30, 11];

          console.log(letters.sort());   
          // returns ['a', 'b', 'c', 'd', 'e']
          console.log(numbers.sort());   
          // returns [1, 11, 3, 30, 5]
        
          function sortNumbers(a, b) {
            return a - b;
          }

          const ascendingNums = numbers.sort(sortNumbers);
        
          function desendingOrder(a, b) {
            if (a > b)
              return -1;
            if (a < b)
              return 1;
            return 0;
          }

          const descendingNums = numbers.sort(desendingOrder);
        
          let cities = [
            "Pessac", 
            "Marseille", 
            "Ribeauvillé", 
            "Périgueux", 
            "Saint-Lô"
          ];

          console.log(cities.sort()); // Leaves cities out of order

          cities.sort((a, b) => {
            return a.localeCompare(b);
          });

          console.log(`Sorted by locale: ${join(", ")}`);

Array.forEach();

The forEach() method is used to run a function on each element of an array.

Unlike map() and reduce(), forEach() cannot be chained and always returns undefined.

forEach() cannot be used with break, continue, or asynchronous commands like promises and async/await.

forEach() does not modify the array on which it is called, but the array can be altered by the callback function.

Syntax:

          arrayObject.forEach(callback[,thisArg]);

Callback Syntax:

  • currentElement: element that is passed in from the array
  • index: the current index of the currentElement
  • array: the original array that is passed through map
          function callback(currentElement, index, array) {...}

Examples:

          const songs = ["song1", "song2", "song3", "song4", "song5"];

          let forExt = [];
          let forEachExt = [];

          for (let i = 0; i < songs.length; i++) {
            forExt.push(` ${songs[i]}.mp3`);
          }

          songs.forEach((item) => {
            forEachExt.push(` ${item}.mp3`);
          });

          console.log(songs);
          console.log(`For loop:${forExt}`);
          console.log(`forEach:${forEachExt}`);
